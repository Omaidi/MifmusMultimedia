# Panduan Upload Manual ke GitHub (Karena Git Tidak Terdeteksi)

Saya mencoba menjalankan perintah upload otomatis, namun **Git tidak ditemukan** di laptop Anda. Tidak masalah, Anda bisa melakukannya secara manual dengan sangat mudah:

## Cara Upload Manual (Drag & Drop)

1. **Buka Repository Anda**:
   Kunjungi link yang Anda berikan: [https://github.com/Omaidi/MifmusMultimedia](https://github.com/Omaidi/MifmusMultimedia)

2. **Mulai Upload**:
   - Jika repository masih kosong, klik link kecil bertuliskan **"uploading an existing file"**.
   - Jika sudah ada isinya, klik **Add file** > **Upload files**.

3. **Pilih File**:
   - Buka folder `d:\TWIBBON` di laptop Anda.
   - Blok/Select semua file yang ada di situ (`index.html`, `style.css`, `script.js`, dan folder `assets`).
   - **Tarik (Drag)** semua file tersebut ke kotak yang ada di halaman GitHub.

4. **Simpan (Commit)**:
   - Tunggu proses upload selesai.
   - Di bagian bawah, ada kotak bertuliskan "Commit changes".
   - Klik tombol hijau **Commit changes**.

## Langkah Terakhir: Aktifkan Website
5. Klik tab **Settings** (ikon gerigi) di repository Anda.
6. Pilih menu **Pages** di sebelah kiri.
7. Di bagian **Build and deployment** > **Branch**, ubah dari `None` menjadi `main`.
8. Klik **Save**.

Tunggu sebentar, lalu refresh halaman. Link website Anda akan muncul!
